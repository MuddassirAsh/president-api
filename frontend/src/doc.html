Documentation

Below you'll find examples using Fetch API but you can use any other http library out there.

Important Notes

  1. Default rate limit is 60 per hour.
  2. Default number of quotes returned from query endpoints is 10.
  3. To view the list of presidents available, click here.
  4. If you don't find the president you are looking for then submit a request here.

Get a random quote

fetch('https://prezidental.xyz/api/random')
      .then(response => response.json())
      .then(quote => console.log(quote))
{
    "quote" : "...",
    "author": "..."
}

Get a random quote by name

fetch('https://prezidental.xyz/api/random/kennedy')
      .then(response => response.json())
      .then(quote => console.log(quote))
{
    "quote" : "...",
    "author": "John F. Kennedy"
}
